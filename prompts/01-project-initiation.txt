---------------------------------------
PROJECT OBJECTIVE
---------------------------------------
AutoResolve is an autonomous incident response system. It receives production alerts from Prometheus. It gathers logs, metrics and traces through Model Context Protocol (MCP).

Multiple AI agents collaborate to:
• understand the incident
• determine probable root cause
• retrieve similar incidents
• recommend remediation
• optionally generate infrastructure/code fixes
• create GitHub Pull Requests
• notify Slack
• generate incident reports

The architecture must emphasize:
• AI-Augmented Engineering
• Agentic AI
• Autonomous SDLC
• Secure AI
• Enterprise Reliability
• Production Readiness
